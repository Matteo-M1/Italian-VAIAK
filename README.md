# Italian-VAIAK

This repository contains materials related to the validation of the Italian version of the VAIAK questionnaire.

**Contents**
- Questionnaires (.qsf format) : Four questionnaires in Italian format, ready to be imported into Qualtrics:
    -  VAIAK-R
    -  Revised-AFS
    -  BICB
    -  CAQ
- 01.VAIAK.R : R script used to analyze the data collected through the VAIAK questionnaire. Part C is specifically designed to compute and properly evaluate the score of the written responses in Italian.
- Database Italian VAIAK validation (2025) : Database used for the validation of the Italian questionnaire. Each sheet corresponds to a different questionnaire, while demographic information is stored in the first sheet.

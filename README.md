# Italian-VAIAK

The repository contains: 
- 01.VAIAK.R is the script written in R to analyse the data collected for the VAIAK questionniare. Part C in particular is necessary to properly assess the score of the written answer in **ITALIAN**
- 4 questionnaires in .qsf format (VAIAK-R, Revised-AFS, BICB, CAQ) in italian. The file can be uploaded to qualtrics
- The database used for the validation of the italian questionnaire. Each sheets are the results of the different questionnaire. The demographical information are contained in the first sheet.
